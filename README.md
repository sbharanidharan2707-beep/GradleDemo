# Task 3 - Java Application using Gradle

## Objective
To automate the build process of a Java application using Gradle.

## Tools Used
- Java JDK 24
- Gradle 9.6.1
- Git
- GitHub
- Windows 11

## Project Description
This project demonstrates how to create, build, and run a Java application using Gradle. Gradle automates the build process, manages dependencies, and simplifies project execution.

## Steps Performed
1. Installed Java JDK and Gradle.
2. Created a new Gradle Java application using:
   ```bash
   gradle init
   ```
3. Built the project using:
   ```bash
   gradle build
   ```
4. Executed the application using:
   ```bash
   gradle run
   ```

## Output

